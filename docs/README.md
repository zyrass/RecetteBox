# Documentation de RecetteBox

Bienvenue dans le centre technique de **RecetteBox**. Cette documentation est organisée de manière séquentielle pour vous accompagner de l'installation jusqu'au déploiement de fonctionnalités avancées.

---

## Structure du Répertoire

### [00-configuration](./00-configuration/)
Guides d'installation et de préparation de l'environnement de développement selon votre système d'exploitation.
*   Fichiers : windows11.md, macos.md, linux.md.

### [01-cursus](./01-cursus/)
Le cœur de l'apprentissage. Chaque phase introduit un concept majeur de l'écosystème Laravel moderne (TALL stack).
*   Fichiers : 01-squelette.md, 02-modele.md, 03-livewire.md.
*   À faire : 04-reactivite.md, 05-crud-alpine.md, 06-dashboard.md, 07-finitions.md, 08-tests.md.

### [09-bonus](./09-bonus/)
Modules optionnels pour aller plus loin une fois le cursus principal maîtrisé.
*   Fichiers : 09-authentification.md.

---

> [!TIP]
> Si vous débutez, commencez toujours par le guide correspondant à votre OS dans le dossier `00-configuration`.
